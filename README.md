# Fizgo Monorepo

A monorepo containing both web and mobile applications for Fizgo.

## Apps

- `apps/web` - Next.js web application
- `apps/native` - React Native mobile application

## Tech Stack

- Next.js
- React Native (Expo)
- TypeScript
- Supabase
- tRPC
- Prisma
- Stripe
- shadcn/ui
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development servers
pnpm dev
```
